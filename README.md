# Nutritionist AI Bot
A Telegram bot powered by [grammY](https://grammy.dev/) and the [GitHub Copilot SDK](https://github.com/github/copilot-sdk) that answers nutrition questions by extracting knowledge from the YouTube channel of a famous biologist and nutritionist.

## How It Works

1. A user sends a nutrition-related question to the Telegram bot.
2. The bot uses the Copilot SDK (Claude Opus 4.5) to interpret the question and extract a search topic.
3. Relevant videos are fetched from the YouTube channel via the **YouTube Data API**.
4. Transcripts of the most relevant videos (up to 5) are retrieved using **youtube-transcript-plus**.
5. The AI analyzes the transcripts and replies with a concise, friendly, and actionable answer — without revealing its sources.

Users can send `/reset` to clear their conversation history and start fresh.

## Tech Stack

| Component | Technology |
|-----------|------------|
| Bot framework | [grammY](https://grammy.dev/) |
| AI / LLM | [GitHub Copilot SDK](https://github.com/github/copilot-sdk) (`@github/copilot-sdk`) |
| HTTP server | [Fastify](https://fastify.dev/) |
| YouTube API | [googleapis](https://github.com/googleapis/google-api-nodejs-client) |
| Transcripts | [youtube-transcript-plus](https://www.npmjs.com/package/youtube-transcript-plus) |
| Schema validation | [Zod](https://zod.dev/) |
| Tunnel (dev) | [ngrok](https://ngrok.com/) |
| Language | TypeScript (ESNext / NodeNext) |
| Formatter | [Biome](https://biomejs.dev/) |

## Project Structure

```
src/
  bot.ts          # Telegram bot setup, webhook server (Fastify on port 3000)
  lib/
    index.ts      # Copilot SDK session management, system prompt, tool wiring
    tools.ts      # YouTube Data API + transcript fetching utilities
scripts/
  ngrok.ts        # Dev helper — starts ngrok tunnel and sets Telegram webhook
build/            # Compiled JS output (generated by tsc)
```

## Prerequisites

- **Node.js** >= 24
- **Yarn** (classic)
- A **Telegram Bot Token** (from [@BotFather](https://t.me/BotFather))
- A **YouTube Data API key** (from [Google Cloud Console](https://console.cloud.google.com/))
- A **GitHub Copilot token** (`github_pat_...`)
- *(dev only)* An **ngrok** auth token and domain

## Environment Variables

Create a `.env` file in the project root:

```env
TELEGRAM_BOT_TOKEN=<your-telegram-bot-token>
YOUTUBE_API_KEY=<your-youtube-api-key>
CHANNEL_ID=<youtube-channel-id>
COPILOT_GITHUB_TOKEN=<github-pat-token>
NGROK_AUTH_TOKEN=<ngrok-auth-token>
NGROK_DOMAIN=<ngrok-domain>
```

## Getting Started

### Install dependencies

```bash
yarn install
```

### Development

Starts the bot in watch mode with an ngrok tunnel for webhook delivery:

```bash
yarn start
```

### Build

Compile TypeScript to JavaScript:

```bash
yarn build
```

### Format

```bash
yarn format
```

## Docker

A multistage Dockerfile is included:

- **Stage 1 (builder)** — installs all dependencies and compiles TypeScript.
- **Stage 2 (runner)** — installs only production dependencies, adds the GitHub Copilot CLI globally, and runs the compiled app.

```bash
# Build the image
docker build -t nutritionist-ai-bot .

# Run the container
docker run -p 3000:3000 --env-file .env nutritionist-ai-bot
```
